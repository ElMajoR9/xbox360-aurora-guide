# Outils PC recommandés

Cette page liste les logiciels PC utiles pour gérer ta Xbox 360 modifiée au quotidien.
Tous les outils listés ici sont gratuits et légaux.

---

## FTP & Gestion de fichiers

### FileZilla
Le client FTP de référence. Gratuit, open source, disponible sur Windows, macOS et Linux.

- **Utilisation** : Transfert de fichiers entre ton PC et ta console
- **Téléchargement** : [filezilla-project.org](https://filezilla-project.org/)
- **Configuration type** : Hôte = IP de la console, Port = 21, User = xbox, Pass = xbox

---

### WinSCP *(Windows uniquement)*
Alternative à FileZilla avec une interface bi-panneau plus intuitive pour certains.

- **Utilisation** : FTP, avec une interface ressemblant à l'Explorateur Windows
- **Téléchargement** : [winscp.net](https://winscp.net/)

---

## Édition de fichiers

### Visual Studio Code
Éditeur de texte moderne pour modifier les fichiers de configuration (`.ini`, `.xml`, `.json`).

- **Utilisation** : Modifier `launch.ini` (DashLaunch), configs Aurora, scripts
- **Téléchargement** : [code.visualstudio.com](https://code.visualstudio.com/)

---

### Notepad++ *(Windows uniquement)*
Alternative légère à VS Code pour l'édition de fichiers texte simples.

- **Téléchargement** : [notepad-plus-plus.org](https://notepad-plus-plus.org/)

---

## Images disque & Extraction

### God2Iso / Iso2God
Convertisseurs entre les formats d'image disque Xbox 360.

!!! info "Usage légal"
    Ces outils sont utiles pour gérer des copies de **tes propres jeux physiques**.
    L'usage de ces outils sur des jeux que tu ne possèdes pas est illégal.

---

### 7-Zip
Gestionnaire d'archives universel. Utile pour décompresser les plugins, thèmes et utilitaires téléchargés.

- **Téléchargement** : [7-zip.org](https://www.7-zip.org/)

---

## Outils réseau

### Advanced IP Scanner *(Windows)*
Scan ton réseau local pour trouver l'IP de ta Xbox 360 sans aller fouiller dans les menus Aurora.

- **Téléchargement** : [advanced-ip-scanner.com](https://www.advanced-ip-scanner.com/)
