# Compatibilité des plugins

Cette page liste les plugins connus et leur statut de fonctionnement sur **Aurora 0.7b.2**,
la version de référence en 2026.

!!! note "Page contributive"
    Cette liste est maintenue par la communauté. Si tu as testé un plugin non listé,
    ou si le statut d'un plugin a changé, [ouvre une issue ou une PR sur GitHub](../contribuer.md).

## Légende

| Icône | Signification |
